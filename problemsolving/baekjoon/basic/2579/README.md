## `K`번째 계단에 착지했다고 할 때
계단마다 두 가지 값을 저장한다.

### Case 1. `K-1`번째 계단에 착지했을 때의 값
`K-1`번째 계단의 Case 2 값을 가져온다.

|계단|K-3|K-2|K-1|K|
|---|---|---|---|---|
|__착지 여부__||❌|✅|✅|

### Case 2. `K-1`번째 계단에 착지하지 않았을 때의 값
`K`번째 계단의 Case 1과 Case 2 중 최댓값을 가져온다.

|계단|K-3|K-2|K-1|K|
|---|---|---|---|---|
|__착지 여부__|✅ 또는 ❌|✅|❌|✅|
